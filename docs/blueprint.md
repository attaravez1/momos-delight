# **App Name**: Azure Bites

## Core Features:

- Shop Information Display: Display shop name, address, contact details, and opening hours retrieved from the Google Maps listing.
- Google Maps Navigation: Implement a 'Get Directions' button that uses Google Maps navigation to guide users from their current location to the shop.
- Image Gallery: Feature a photo gallery displaying real images of the shop and food items, sourced from Google Maps photos and customer reviews.
- About Us Section: An 'About Us' section presenting the shopâ€™s story, specialties, and quality.
- Menu Display: Display a menu with food items, prices, and high-quality photos, and the ability to use a tool powered by generative AI to decide if seasonal ingredients need any edits.
- Customer Reviews Carousel: Display a customer reviews carousel to showcase feedback from Google Maps reviews.
- Contact Section: Implement a contact section with direct links to WhatsApp, call options, and location maps, as well as social media

## Style Guidelines:

- Primary color: Navy Blue (#1E3A8A) for a sense of trust and sophistication.
- Background color: Off-White (#F9FAFB) to create a clean and modern canvas.
- Accent color: Sky Blue (#7DD3FC) to add a vibrant and inviting touch to interactive elements.
- Body font: 'PT Sans' sans-serif font, used for body text.
- Headline font: 'Playfair' serif font, used for all headings and sub-headings.
- Use clean, modern icons in blue tones for navigation and key features.
- Elegant and minimal design with a clean, modern navigation bar. Incorporate smooth scrolling to enhance user experience. 
- Subtle motion animations for text and images, smooth transitions, and parallax effects for a dynamic feel.
